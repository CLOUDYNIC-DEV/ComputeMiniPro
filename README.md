# Compute Mini Pro: The AI Compute Pro

The **Compute Mini Pro** is an ultra-compact, high-performance carrier board designed specifically for the **Rockchip RK3588 CM5 module**. Built for engineers who need **6 TOPS of NPU power** in their pocket, this board breaks the tradition of bulky development kits by packing industrial-grade I/O into a handheld form factor.

> **Engineer's Note:** 
> *This entire 4-layer high-speed board was routed and finalized on an aging Inspiron N5010. It’s proof that grit and clean routing beat high-end workstations every time.*

---

## Hardware Specifications

The carrier board is engineered to unlock the full potential of the **RK3588S / CM5 ecosystem**:

### Core Architecture
*   **SoC Support:** Full compatibility with **RK3588 CM5** modules.
*   **PCB Architecture:** **4-layer stackup** featuring:
    *   Controlled impedance 
    *   Precise length matching for **LPDDR5 / PCIe Gen3** stability.

### High-Speed I/O & Connectivity
*   **1x HDMI 2.1:** Supports up to **8K @ 60fps** output.
*   **1x USB 3.1 Type-C:** High-speed data and integrated **Power Delivery (PD)**.
*   **1x Ethernet Port:** Equipped with the fastest transceiver for **ultra-low latency** networking.

### Storage & Power
*   **Storage:** Onboard **MicroSD slot** + **eMMC support** via the CM5 module.
*   **Power System:** Advanced, robust **power tree** designed for stable delivery under heavy **AI inference loads**.
*   **Connectors:** Dual **high-density B2B (Board-to-Board) connectors** for secure module mating.
